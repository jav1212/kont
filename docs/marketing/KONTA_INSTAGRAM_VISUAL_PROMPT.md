# Konta Instagram Visual Prompt

Use this prompt in ChatGPT Images or another image-generation tool to create premium Instagram visuals for the `Konta.` brand, inspired by the provided reference style.

## Brand non-negotiables

- Brand name: `Konta.`
- The logo is the word `Konta.`
- The text part of the logo must be white or black depending on the background
- The final dot must always use the brand main color: `#FF4A18`
- `#FF4A18` is the primary accent color of the brand

## Master Prompt

```text
Create a premium Instagram brand campaign visual for a software brand called "Konta." inspired by a high-end fintech/product identity system. The composition should feel like a polished creative-agency campaign with clean modular panels, luxury product-marketing aesthetics, and strong visual consistency.

The style should closely match a premium product-brand rollout: elegant, minimal, glossy, modern, and highly art directed. Use a refined visual system with:

- soft rounded rectangular panel layout
- premium product mockup composition
- clean white or very light gray background
- teal, aqua, turquoise, dark petroleum, and deep green gradients
- glossy glass-like reflections
- subtle highlights and modern shadowing
- smooth curved abstract shapes in the background
- minimal but bold typography
- premium startup / fintech / SaaS branding feel
- visually clean grid composition suitable for Instagram
- luxury brand presentation style

Include the "Konta." logo as the main visible brand mark, replacing the reference brand. The wordmark should appear elegant, clean, modern, and high-end, with the exact text:

"Konta."

Logo rule:

- render `Konta` in black or white depending on background contrast
- render the final dot `.` in `#FF4A18`
- preserve this contrast rule consistently across all compositions

The generated composition should feel like a product identity campaign for a premium financial or business software platform. It should look like a curated brand presentation, not a generic ad.

Design language:

- premium fintech identity
- clean Apple-style product presentation
- modern SaaS branding
- glossy but restrained
- high-end minimalism
- commercial but editorial

Possible brand application surfaces to show:

- smartwatch or mobile app screen mockup
- desktop/tablet UI crop
- branded printed card or product card
- social card or promo card

All surfaces should use the Konta. brand identity consistently.

If typography is shown inside the composition, keep it minimal and elegant. Use short lines such as:

- "Konta."
- "One platform."
- "Control your business."
- "Payroll, inventory, documents."
- "Built for growing companies."

Do not overload the layout with text.

Important constraints:

- do not generate a generic corporate stock ad
- do not use random unrelated brand names
- do not clutter the composition
- do not use purple as the main color
- do not make it look cartoonish
- do not make it look like a template marketplace design

Make the final image feel like a premium, modern, visually coherent Instagram campaign for the brand Konta.

Format: square 1:1 for Instagram post
Quality: ultra polished, premium, realistic brand visualization, art directed, clean, cohesive
```

## Optional Variant for a Feed Series

```text
Create a cohesive Instagram campaign series for the brand "Konta." using the same premium visual identity system: rounded modular panels, white/light-gray backgrounds, teal and aqua gradients, dark petroleum accents, glossy reflections, soft curved shapes, minimal typography, and elegant software-brand presentation. Every image should feel like part of the same high-end fintech/SaaS brand system. Replace all branding with "Konta." and keep the visual language premium, clean, and modern.
```

## Usage Notes

- If the model supports reference images, attach your visual reference and use the master prompt.
- If the model supports logo reference upload, include the Konta logo so the wordmark is closer to your real brand.
- If the model struggles with text fidelity, ask for:
  - fewer words
  - larger logo placement
  - one primary headline only
